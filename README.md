# SFDataBase

## Описание

Учебный проект по SQL для MySQL.  
Проект содержит скрипт создания и заполнения базы данных, а также набор SQL-запросов для решения задач по четырем заданиям.

---

База данных представляет собой общим хранилещем, которое содержит все таблицы для заданий в одной схеме:

- транспортные средства;
- автомобили и гонки;
- отели и бронирования;
- сотрудники, отделы, проекты и задачи.

---

| Таблица | Задание | Краткое описание |
|---|---:|---|

| `Classes` | 2 | Справочник классов автомобилей. Хранит тип, страну, число дверей, объем двигателя и массу. |

| `Hotel` | 3 | Справочник отелей. Содержит идентификатор, название и местоположение отеля. |

| `Tasks` | 4 | Таблица задач. Содержит название задачи, исполнителя и проект, к которому относится задача. |

## Решенные задачи
### Задание 1. Транспортные средства
* Задача 1 — поиск спортивных мотоциклов по мощности и цене.
* Задача 2 — объединенный поиск автомобилей, мотоциклов и велосипедов по заданным критериям.
### Задание 2. Автомобили и гонки
* Задача 1 — поиск автомобилей с лучшей средней позицией в каждом классе.
* Задача 2 — поиск автомобиля с лучшей средней позицией среди всех автомобилей.
* Задача 3 — определение классов с лучшей средней позицией и вывод автомобилей этих классов.
* Задача 4 — поиск автомобилей, чья средняя позиция лучше средней позиции по своему классу.
* Задача 5 — анализ классов и автомобилей с низкой средней позицией.
### Задание 3. Отели и бронирования
* Задача 1 — анализ клиентов с более чем двумя бронированиями в разных отелях.
* Задача 2 — анализ клиентов с более чем двумя бронированиями и суммой расходов более 500 долларов.
* Задача 3 — определение предпочтительного типа отелей для каждого клиента.
### Задание 4. Сотрудники и проекты
* Задача 1 — рекурсивный поиск сотрудников, подчиняющихся Ивану Иванову.
* Задача 2 — рекурсивный поиск сотрудников с подсчетом задач и количества прямых подчиненных.
* Задача 3 — поиск менеджеров с подчиненными и подсчет общего количества подчиненных.

## Требования

### Для запуска проекта необходимо:
* установленный MySQL Server;
* клиент для выполнения SQL-скриптов: MySQL Workbench, или консольный клиент mysql.

## Запуск проекта

* Создайте сервер и запустите MySQL через MySQL Configurator 8.0+
* Подключитесь к серверу через MySQL Workbench 8.0+
* Запустите скрипт DDL+DML.sql, который создает схему 'sfdb' со всеми необходимыми таблицами (сразу с данными)
* Запустите Task-скрипты по всем заданиям.
* Удаление происходит через команду: DROP SCHEMA IF EXISTS sfdb;
